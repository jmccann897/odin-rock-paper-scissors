# odin-rock-paper-scissors
Odin Rock Paper Scissors repository for the Foundations Course: Rock Paper Scissors Project 

## Description

A simple Rock Paper Scissors JavaScript game updated from being played in the console to clicking UI buttons.

The user can click the **rock**, **paper**, or **scissors**  buttons to play a round against the computer. The computer randomly selects its move each round.

After each round:
- The player's choice and the computer's choice are displayed below the options.
- The result of the round (win, lose, or draw) is shown.
- The running score for the player, computer, and draws is updated.

The game continues until either the user or computer reaches a score of 5. The user can click '**x**' to exit at any time.
