# odin-rock-paper-scissors
Odin Rock Paper Scissors repository for the Foundations Course: Rock Paper Scissors Project 

## Description

A simple Rock Paper Scissors JavaScript console game.

The user is prompted to enter **rock**, **paper**, or **scissors** to play a round against the computer. The computer randomly selects its move each round.

After each round:
- The player's choice and the computer's choice are displayed in the browser console in developer tools.
- The result of the round (win, lose, or draw) is shown.
- The running score for the player, computer, and draws is updated.

The game continues in a loop until the user enters '**x**' to exit.
